Usage

   
 dlabpro FSG.xtp <command> <cfgfile> [options]
 dlabpro FSG.xtp help [<command>]
 
Argument/OptionDescription
Arguments
$1: command Command, one of the following:
 
trn - Language model training

or "help"
$2: cfgfile UASR configuration file
 
Options
-c Use colors for screen protocol
-h, --help Display help and exit
-P<key>=<value> Define or override configuration key-value pairs
-n Write protection. If set, no output files will be written.
-v<n> Verbose level
-v1   minimal screen protocol (default)
-v2extended screen protocol
-v3even more screen protocol, program may break
There are further, individual options to the commands.
 

Configuration

FSG.xtp is configured through a UASR configuration file (parameter $2). The configuration file is a Java properties file containing key-value-pairs. The common configuration keys are described with the utility script cfg.itp (see there). Some commands use dedicated configuration keys which are described with the command. Dedicated keys may be placed in the configuration file just like all other keys. Any key (common and dedicated) may be specified or overwritten through the command line using -P<key>=<value> options:

 dlabpro FSG.xtp trn myconfig.cfg -Plm.ngram.cutoff=1

-P options always overwrite the configuration file. The heading "uasr." of the keys may be omitted in -P options.

For further information

  • see documentation of util/cfg.itp to learn more about UASR's organization of databases.
  • see documentation of -CFG_get_path to learn how UASR determines paths to the various data sets.
  • see documentation of -CFG_get_flist to learn how UASR obtains file lists of the various data sets.

Function -LM_load_wordlist Functions Script Package Keyword Index Function Index Home
    Signature :    -LM_load_wordlist(sLexFile,idWls)
    Source :    FSG.xtp(202)

Description

Loads a word list from a lexicon file. The function uses {@link -LX_import} to load the lexicon. The function will assure that the generated word list contains the pseudo-words "" (out-of-vocabulary), "_" (sentence start), and "."(sentence end). Add pseudo-words which are not yet contained in the lexicon will be added to the word list.

Parameters

   string    sLexFile   The relative or fully qualified name of the lexicon file.
   data    idWls   Filled with the word list.
Function -LM_gather_wordlist Functions Script Package Keyword Index Function Index Home
    Signature :    -LM_gather_wordlist(iF,nWrdCutoff,idWls)
    Source :    FSG.xtp(228)

Description

Gathers a word list from transliteration files. The function will add the pseudo-words "" (out-of-vocabulary), "_" (sentence start), and "." (sentence end) to the list. The transliteration files must not contain any of these pseudo-words by definition!

Parameters

   file    iF   The file list.
   number    nWrdCutoff   Words with counts less or equal the cut-off will be discarded and replaced by the out-of-vocabulary pseudo word "".
   data    idWls   Filled with the word list.
