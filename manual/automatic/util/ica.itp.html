Function -ICA_ite Functions Script Package Keyword Index Function Index Home
    Signature :    -ICA_ite(iF,iFI,nI,idWI)
    Source :    util/ica.itp(158)

Description

Performs one iteration of the incremental fast ICA algorithm.

Parameters

   file    iF   File list defining a set of feature files to gather the ICA statistics from
   object    iFI   Feature information object
   number    nI   Iteration counter
   data    idWI   Orthonormal unmixing matrix, will be overwritten by result of the iteration

Return value

TRUE if another iteration should be performed, FALSE if no further iterations should be performed. The latter occurs if a) the algorithm converged or b) the maximal number of iterations was reached
