Description

Loads the data files produced by the recognizer from the locations committed thorugh the command line arguments.

Parameters

   integer    nFArg   One-based index of first command line argument, standard is 1.
   object    iSI   Filled with the session info, can be NULL if not needed.
   fst    itRec   Filled with the main recognition result, can be NULL if not needed.
   fst    itRef   Filled with the reference recognition result, can be NULL if not needed.
   data    idNld   Filled with the frame-by-Gaussian neglog. density array, can be NULL if not needed.
   data    idSig   Filled with the input speech signal, can be NULL if not needed.
   fst    itFvr   Filled with the recognized semantics (feature-value relation), can be NULL if not needed.
Function -RRES_process Functions Script Package Keyword Index Function Index Home
    Signature :    -RRES_process(itRes,idNld,idPhnTab)
    Source :    util/rec_postproc.itp(74)

Description

Processes a recognition result transducer. The function stips uninteresing components, removes epsilon input symbols and adds phoneme label names.

Parameters

   fst    itRes   The recognition result transducer (consecutive path though the recognition network).
   data    idNld   The neglog likelihood matrix (records: feature vectors, components: GMMs).
   data    idPhnTab   The phoneme table, phoneme labels are expected in component 0.

Return value

A data object containing the processed path.
