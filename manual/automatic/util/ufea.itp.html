Description

Universal primary feature analysis This function preforms the universal primary feature analysis.

Parameters

   data    idSig   Signal to be analyzed, may contain multiple signal channels, may contain one or multiple label components.
   data    idPM   Pitch marks for pitch synchronous analysis, may be NULL
   data    idFea   Will be filled with the feature output vectors. Labels will be copied from idSig.
   data    idDummy   Only for compatible with imaginary data object in dlabpro's feature processors. Should be NULL.

Function -UPFA_init Functions Script Package Keyword Index Function Index Home
    Signature :    -UPFA_init
    Source :    util/ufea.itp(212)

Description

Initialize universal primary feature analysis This function creates the object .__FEA_iPfa and sets up the methods -analyze and -synthesize. Additionally it reads all required configuration keys. Finally the object is compatible to feature analysis/synthesis processor instances in dlabpro.
