Function -DATA_diff Functions Script Package Keyword Index Function Index Home
    Signature :    -DATA_diff(idX,idRef)
    Source :    util/data.itp(165)

Description

Compares two data tables and returns the normalized absolute difference. The function computes the sum over abs(idX[i,k]-idRef[i,k])/abs(idRef[i,k]) for all 0<=i<idRef.dim and 0<=k<idRef.nrec.

Parameters

   data    idX   Test table
   data    idRef   Reference table

Return value

The normalized absolute difference
