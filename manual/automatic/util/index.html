Format of Data Instances

Signal instances
comp. type name description
0 * float or double user defined Signal data
1 ... * string user defined Symbolic labels
- Record axis: time [ms]
- Data range: -1 ≤ s(k) ≤ 1

* Component index is significant, names do not matter.
° Component name is significant, index does not matter.
