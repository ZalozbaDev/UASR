Function -STAT_quota Functions Script Package Keyword Index Function Index Home
    Signature :    -STAT_quota(n,K)
    Source :    util/stat.itp(62)

Description

Computes a quota with 95% Clopper-Pearson interval.

See R. Hoffmann, M. Wolff: Intelligente Signalverabeitung 2 - Signalerkennung, Springer-Vieweg, 2015. pp. 132-136.

Parameters

   number    n   No. of hits.
   number    K   Sample size.

Return value

A data table with the components "q" (quota), "q+" (difference to upper bound of Clopper-Pearson interval), "q-" (difference to lower bound of Clopper-Pearson interval), "n" (n), and "K" (K).
