Function -SAVE Functions Script Package Keyword Index Function Index Home
    Signature :    -SAVE(sFile,sFormat,iObject)
    Source :    util/uasr.itp(566)

Description

Saves an object to a file regarding the global write-protect flag. The global write-protect flag is stored in the variable .__CFG_bNwr and toggled through the command line option "-n".

Parameters

   string    sFile   The relative or fully qualified name of the file to save to.
   string    sFormat   The file format. See documentation of class file for a list of available formats. Can be NULL in which case the object will be saved in the dLabPro standard file format (zipped XML or DN3).
   object    iObject   The object to save (cannot be NULL).

Return value

0 if the object was actually saved, 1 if write-protection is active, and -1 if an error occurred while trying to write the file.
