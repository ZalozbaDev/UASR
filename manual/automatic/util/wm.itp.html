Function -WM_addtrans Functions Script Package Keyword Index Function Index Home
    Signature :    -WM_addtrans(nIni,nTer,nAct,bFinal,itWm)
    Source :    util/wm.itp(119)

Description

Adds a transition to a world automaton. If the transition already exists and the transition table of itWm contains a reference counter component, the function will increment the reference counter.

Parameters

   integer    nIni   The initial state of the transition.
   integer    nTer   The terminal state of the transition.
   integer    nAct   The action.
   boolean    bFinal   Indicates whether the terminal state is final.
   fst    itWm   The world automaton. If empty, the function will create a new world model in unit 0.

Return value

The zero-based transition index of the added/updated transition.

Function -WM_qlearn Functions Script Package Keyword Index Function Index Home
    Signature :    -WM_qlearn(idAct,nDefQ,nDctQ,nXIte,itWm,itIwm)
    Source :    util/wm.itp(397)

Description

Simple Q-learning. The function interacts with the world model itWm only by invoking -WM_setrandomstate and -WM_act. These functions may be overloaded to implement other worlds.

Parameters

   data    idAct   A list of actions which can be performed in the world.
   double    nDefQ   Initial Q-value, 3 is a reasonable choice.
   double    nDctQ   Q discount factor, 0.9 is a reasonable choice.
   int    nXIte   Number of iterations.
   fst    itWm   The world model in unit 0.
   fst    itIwm   Filled with the inner world model of the cognitive agent after Q-learning.

Return value

void
