Function -RRES_perframe Functions Script Package Keyword Index Function Index Home
    Signature :    -RRES_perframe(itRes,idNld,idPhnTab)
    Source :    util/rres.itp(207)

Description

Computes the per-frame recognition result. The function stips uninteresing components, removes epsilon input symbols and adds phoneme (component "~PLB") and output (component "~OLB") label names.

Parameters

   fst    itRes   The recognition result transducer (consecutive path though the recognition network).
   data    idNld   The neglog likelihood matrix (records: feature vectors, components: GMMs).
   data    idPhnTab   The phoneme table, phoneme labels are expected in component 0.

Return value

A data object containing the processed path.

Function -RRES_collect Functions Script Package Keyword Index Function Index Home
    Signature :    -RRES_collect(idSig,idRpf,itFvr,idPhnTab,sDirOut)
    Source :    util/rres.itp(394)

Description

Collects the recognition result into a speech database. The function will create/update the following folder structure 
   <sDirOut>
   |-- fvr
   |   |-- <TID>.fst - Recognized semantics (feature-value relation)
   |   '-- ...
   |-- par
   |   |-- <TID>.par - Partitur file containing MAU and ORT tiers
   |   '-- ...
   |-- sig
   |   |-- <TID>.wav - Signal file
   |   '-- ...
   '-- collect.flst  - File list including the recognition result (word sequence or FVR string)

where <TID> denotes the unique turn ID (time stamp).

Parameters

   data    idSig   The speech signal.
   data    idRpf   The per-frame recognition result as returned by RRES_perframe.
   fst    itFvr   The recognized semantics (feature-value relation represented by an FST), can be NULL or emtpy. The transliteration is expected as FVR comment string in itFvr.ud.rtext.
   data    idPhnTab   The phoneme table, phoneme labels are expected in component 0.
   string    sDirOut   Output directory for collected data.
