Loads a Partitur file.
Function -PAR_load Functions Script Package Keyword Index Function Index Home
    Signature :    -PAR_load(sFile,sFilter,idPar)
    Source :    util/par.itp(66)

Description

Loads a Partitur file. A Partitur file is a text file containing different data tiers marked by an identifier "<TID>:" at the beginning of each line. Blank lines and lines starting with a hash tag (#) are ignored.

Example

 Comment
FOO: what ever ...
 
GRM: (S) H hundert (A) und (B) ZE
GRM: (A) (F)
GRM: (A) (B)
GRM: (S) (B)
 
BAR: what ever ...

Includes

The special tier marker "INC:" allows including another Partitur file at the position of the include marker. Includes can be nested. Cyclic includes will be detected and suppressed.

FOO: what ever ...
 
 Include another file
INC: otherfile.par
 
BAR: what ever ...

Include supports absolute and relative path names. "Relative" means relative to the primary Partitur file specified through the sFile argument.

Parameters

   string    sFile   Path of Partitur file to be loaded.
   string    sFilter   Tier filter (consider only lines starting with the given string), "" or NULL will read all lines.
   data    idPar   Will be filled with Partitur tier(s).

Return value

TRUE if successfull, FALSE otherwise
