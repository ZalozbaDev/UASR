Function -Into_get Functions Script Package Keyword Index Function Index Home
    Signature :    -Into_get(sFile,idInv,idInto)
    Source :    synth/into.itp(54)

Description

Obtains an into data object for one turn. There are different operation modes defined by configuration key uasr.into: file Into is loaded form a file named $[sFile].into in $[sDirInto]. grm Into is generated using the grammar from $[sDirGrm]/syn.grm. The transcription in $[sDirTrans]/$[sFile].trans is used. The generated into is saved to $[sDirLog]/$[sFile].into for debugging purposes.

Parameters

   string    sFile   Turn ID
   data    idInv   Diphone inventory used for phonem length calculation in grm mode. May be NULL in file mode.
   data    idInto   Will be filled with the into data

Access to Configuration and Global Instances

   sDirInto    read 
   sDirTrans    read 
   sDirGrm    read 
   sDirLog    read 
   <uasr.into>    read
