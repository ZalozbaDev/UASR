Function -HMM_vocode Functions Script Package Keyword Index Function Index Home
    Signature :    -HMM_vocode(idSrc,iFI,itHMM,idGss,idFea)
    Source :    synth/am.itp(61)

Description

HMM vocoding. Computes the optimal HMM path(s) by aligning the labelled feature vector sequence idSrc on the respective HMMs and resynthesizes the feature vector sequence from that paths

Parameters

   data    idSrc   Primary feature vector sequence to be HMM-vocoded
   object    iFI   Feature information object (see function -FEA_info)
   hmm    itHMM   Hidden Markov Model to vocode with
   data    idGss   will be filled with the Gaussian index sequence used to synthesize idFea, may be NULL
   data    idFea   will be filled with the vocoded primary feature vector sequence, may be NULL

Function -HMM_layer_map Functions Script Package Keyword Index Function Index Home
    Signature :    -HMM_layer_map(idLyrMap,itHMM)
    Source :    synth/am.itp(528)

Description

EXPERIMENTAL; creates a Gaussian/per-layer statistics index map for HMM synthesis. The function creates
  • a map assigning each HMM state and each HMM transition the index of their per-layer statistics (component "~PLS" of state and transition lists) and
  • a map assigning each per-layer statistics the set of Gaussians for which this statistics accounts (idLyrMap).

Parameters

   data    idLyrMap   Will be filled with a Gaussian/per-layer statistics index map, may be NULL.
   hmm    itHMM   HMM instance, the function will overwrite the contents of the components named "~PLS" in the state and transition tables. If no such components exist, they will be created
