Usage

 dlabpro CFG.xtp <command> [arguments] [options]
 dlabpro CFG.xtp help [<command>]
 
Argument/OptionDescription
Arguments
$1: command Command, one of the following:
 
dmp - Dump a configuration to stdout
 dlabpro CFG.xtp dmp <cfgfile> [options]
inf - Display information about a configuration
 dlabpro CFG.xtp inf <cfgfile> [options]
ref - Create reference recognition results entries
 dlabpro CFG.xtp ref <evalfile> [outfile] [options]

or "help"
$2: cfgfile UASR configuration file
$2: evalfile HMM/SVM evaluation result file (hmm_eval.data.xml or svm_eval.data.xml in log directory)
$3: outfile Output file (ASCII, key-value pairs, to be included into a configuration file); if omitted the output goes to stdout
 
Options
-h, --help Display help and exit
-P<key>=<value> Define or override configuration key-value pairs
-v<n> Verbose level
-v1   minimal screen protocol (default)
-v2extended screen protocol
-v3even more screen protocol, program may break
 
End of user documentation.
