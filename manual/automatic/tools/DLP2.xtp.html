General     Signature :    DLP2 $1 $2 $3
    Author :    Matthias Wolff

Synopsis

dLabPro data file conversion tool

Arguments

     $1   Input file
     $2   Output file format (optional)
     $3   Output file (optional)

Usage

 dlabpro DLP2.xtp <infile> [<format>] [<outfile>] [options]
 
Argument/OptionDescription
Arguments
$1: infile dLabPro data file to be converted
$2: format Target file format, one of the following1):
 
ascii Tab stop separated ASCII text
au SUN AU audio1)
csv Comma separated values file
csv_de Semicolon separated values file
esps_labESPS label file
pm Period marker file (binary)
pm.txt Period marker file (Wavesurfer)
raw Raw binary file
wav MS wave audio1)

The argument is optional. If omitted the file contents will be dumped to the console.
$3: outfile Target file. The argument is optional. If omitted the output file name will be "<infile>.<format>".
 
Options
-h, --help Display help and exit
1) For a complete list see documentation of dLabPro class file
2) Input file will be filtered
 

Examples

 dlabpro DLP2.xtp --help                      ( display help                      )
 dlabpro DLP2.xtp foo.dn3                     ( screen dump foo.dn3               )
 dlabpro DLP2.xtp signal.dn3 wav signal.wav   ( convert signal.dn3 to a wave file )

 

End of user documentation.
