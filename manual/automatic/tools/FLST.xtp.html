FLST.xtp exc Script Package Keyword Index Function Index Home
    Signature :    FLST.xtp exc <exclflst> <flstfile> ... [options]
    Synopsis :    Exclude entries from file list(s)

Description

Seeks all entries of a file list <exclflst> in one or several other file lists, removes these entries and saves the modified lists.

Start and Command Line Options

The program is started as follows:

 dlabpro FLST.xtp exc <exclflst> <flstfile> ... [options]
 
Argument/OptionDescription
Arguments
exclflst list of entries to exclude
flstfile file list(s) file to be processed
 
Options
-n do not write output, just say what to do

The standard command line options are accepted (see section Usage above).

Examples

 dlabpro FLST.xtp exc exclude.flst *.flst

Exclude entries in exclude.flst from all files named *.flst.

 dlabpro FLST.xtp exc exclude.flst *.flst -n

Just tell what to do for excluding entries in exclude.flst from all files named *.flst.

FLST.xtp mrg Script Package Keyword Index Function Index Home
    Signature :    FLST.xtp mrg <flist1> [<flist2> ...] [options]
    Synopsis :    Merge file lists

Description

Merges and/or shuffles file lists.

Start and Command Line Options

The program is started as follows:

 dlabpro FLST.xtp mrg <flist1> [<flist2> ...] [options]
 
Argument/OptionDescription
Arguments
flist1 first file list
flist2 second file list (optional)
... optionally more file lists
 
Options
-Pi=TRUE/FALSE If TRUE, the file lists will be interleaved (default: concatenate lists)
-Po=<filename> Store resulting file list in <filename> (default: print at stdout)
-Pr=TRUE/FALSE If TRUE, shuffle input file lists.

The standard command line options are accepted (see section Usage above).

Example

 dlabpro FLST.xtp mrg foo.flst bar.flst -Pi=TRUE -Po=result.flst

Interleaves the file lists foo.flst and bar.flst without shuffling and stores the resulting file list into result.flst.

FLST.xtp spl Script Package Keyword Index Function Index Home
    Signature :    FLST.xtp spl <flstfile> [options]
    Synopsis :    Filter and/or randomly split file list

Description

Filters a file lists for entries with a particular label (provided the file list contains labels in a second coloumn), randomly selects a sepcified quantity of the remaining entries, saves the selected entries to one new file list and the unselected entries to another new file list. This program is useful to disjoint create training, test and evaluation sets.

Start and Command Line Options

The program is started as follows:

 dlabpro FLST.xtp spl <flstfile> [options]
 
Argument/OptionDescription
Arguments
flstfile file list file to be processed
 
Options
-Pl=<label> Filter for label (2nd column in file list). Optional; if omitted all entries will be processed regardless of their labels.
-Pn=<number> Number of entries to pick randomly. Optional; if omitted all entries remaining after filtering will be written to the first output file. The option is mutually exclusive with -Pp.
-Pp=<amount> Amount of entries to pick randomly (0 for 0 % through 1 for 100 %). Optional; if omitted all entries remaining after filtering will be written to the first output file. The option is mutually exclusive with -Pn.
-Pr=TRUE/FALSE If TRUE, shuffle output file lists (default: FALSE).
-Po1=<filename> First output file. Optional; if omitted the output goes to stdout, if an empty filename is committed the output will be discarded.
-Po2=<filename> Second output file. Optional; if omitted the output goes to stdout, if an empty filename is committed the output will be discarded.

The standard command line options are accepted (see section Usage above).

Example

 dlabpro FLST.xtp spl all.flst -Pl=Z00 -Pn=100 -Po2=

Filter entries labelled "Z00" from all.flst, randomly pick 100 entries, dump picked entries at stdout and discard all others.
