Synopsis

Database check-up tool

Arguments

     $1   Command or 'help'
     $2   Setup file or, if 1st arg. is 'help', command

Usage

   
 dlabpro CHK.xtp <command> <cfgfile> [options]
 dlabpro CHK.xtp help [<command>]
 
Argument/OptionDescription
Arguments
$1: command Command, one of the following:
 
fea - Check primary feature vector files
fls - Check set of file lists
sig - Check signal files

or "help"
$2: cfgfile UASR configuration file
 
Options
-h, --help Display help and exit
-P<key>=<value> Define or override configuration key-value pairs
-v<n> Verbose level
-v1   minimal screen protocol (default)
-v2extended screen protocol
-v3even more screen protocol, program may break
 
CHK.xtp fea Script Package Keyword Index Function Index Home
    Signature :    CHK.xtp fea <cfgfile> [options]
    Synopsis :    Check primary feature vector files

Description

Checks the primary feature vector files of the database.

Start and Command Line Options

The program is started as follows:

 dlabpro CHK.xtp fea <cfgfile> [options]
 
Argument/OptionDescription
Arguments
cfgfile UASR configuration file

The standard command line options are accepted (see section Usage above).

Example

 dlabpro CHK.xtp fea myconfig.cfg

Checks the primary feature vector files of the myconfig.cfg UASR setup.
